# 雷电战机 (Thunder Warrior)

一款用纯 HTML5 Canvas 制作的竖版射击游戏。

## 特色

- 🇨🇳 **中国战机**: 歼-10、歼-16、歼-20、歼-35
- 🎯 **积分系统**: 击落不同敌机获得 100-2000 分
- ❤️ **血量系统**: 起始 10000 血量，顶部血条显示
- ⬆️ **升级系统**: U 键升级，血量上限 +2000

## 操作

| 操作 | 功能 |
|------|------|
| 鼠标移动 | 控制飞机位置 |
| 自动射击 | 无需按键 |
| U 键 | 升级面板 |

## 敌机类型

| 类型 | 血量 | 积分 |
|------|------|------|
| 轻型战机 (F-16) | 1 | 100 |
| 战斗机 (F-15) | 2 | 200 |
| 轰炸机 (Su-27) | 5 | 500 |
| 母舰 (MiG-29) | 15 | 2000 |

## 运行

直接用浏览器打开 `index.html` 即可游戏。
